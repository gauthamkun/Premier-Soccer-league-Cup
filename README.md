#### To Run The Project:
1. Run the Spring Boot Java File - Go to /src/main/java/io/soccerapp/premiersoccerleagueapp
2. Run the React App - Go /src/frontend
3. "npm install"
4. "npm start"
5. The Application will be running on "http://localhost:3000/"
